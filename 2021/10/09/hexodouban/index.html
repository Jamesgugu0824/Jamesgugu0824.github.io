关于hexo douban爬不到数据的问题

发表于 2021-10-09

问题：hexo douban

hexo douban 运行结束之后，结果如下：

原因：

当前使用的node版本太高了，降低node版本即可

解决方法：

首先卸载目前这个版本的node， 然后重新去官网，找到版本12下载即可，我之前是14.16 下载12.22.6

下载的时候选择msi，下载完直接安装就可以用。

注意：安装时最好在你原来nodejs的安装目录，这样环境变量也不用重新配了

然后执行hexo clean 再 hexo douban

然后 hexo generate 再 hexo server 就完成了！
